# 📄 PDF & Image Optimizer — Estensione Firefox

Comprime automaticamente **PDF e immagini** prima di caricarli su qualsiasi sito web, direttamente nel browser. Nessun server esterno, tutto in locale.

## ✅ Funzionalità
- Comprime PDF, JPEG, PNG, WebP, BMP, TIFF, AVIF
- Mostra il risparmio ottenuto (es. 2.4 MB → 1.1 MB -54%)
- 3 livelli di compressione: Leggera / Media / Massima
- Funziona su tutti i siti senza configurazione
- Privacy totale: i file non lasciano mai il browser

## 📥 Installazione
Scarica il file `.xpi` e installalo in Firefox tramite:
`about:addons` → ⚙️ → "Installa componente aggiuntivo da file"
